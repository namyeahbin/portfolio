Publishing Guide

Open Graph |

오픈그래피 적용 meta property="og" 적용

Favicon |

5개의 이미지와 1개의 JSON 파일로 파비콘 최적화 32x32 예전 브라우저 svg 최신 브라우저 / 다크모드 지원 apple-touch-icon 180x180 애플기기 192x192 홈스크린 manifest(JSON) PWA 및 안드로이드 기기 512x512 PWA 로딩시 스플래시 스크린용

HTML5 |

W3C기준 meta[name:viewport]설정 header / main / footer를 이용하여 영역 분리 img태그 사용시 alt값 필수 작성 a태그 사용시 빈 링크는 #으로 통일

CSS3 |

첫줄에 인코딩 선언 @charset "UTF-8"; 주석을 이용하여 영역별 css 구분 모든 속성 선언 끝에 세미콜론; 작성

파일명 |

웹페이지는 첫 페이지는 index.html로 통일 알파벳, 숫자, 언더바_ 조합하여 파일명 설정

폴더 구조 |

css 공통으로 연결된 font.css와 menu.css가 담겨있는 폴더 fonts 웹폰트를 제외한 폰트 파일이 담겨있는 폴더 images 공통으로 연결된 이미지가 담겨있는 폴더 script 공통으로 연결된 .js 파일이 담겨있는 폴더

ProblemSolution

다양한 슬라이드 적용 문제

swiper js로 메인썸네일과 상품 슬라이드의 두가지 이상의슬라이드 구현이 되지 않는 문제

스크립트에 각 변수 a와b를 설정하고 적용할 슬라이드에서로 다른 클래스명을 주고 변수에 대입해서 문제 해결

비디오 화면 비율 제어 문제

비디오 화면이 비율에 맞게 들어가지 않는 문제 발생 모바일에서도 동일하게 보여질 수 있도록 width값 100%를 주고 object-fit cover를 주어 문제 해결

Ajax로 필요한 json파일 데이터 출력의 미숙함

url경로설정 오류와 기존 css의 붕괴 구조를 다시 한 번 보고 경로설정을 바르게 하고 기존 html과 css 값에 하나하나 json 데이터 적용

슬라이드 적용의 어려움

JavaScript 배우고 실제 웹사이트에 적용하는 것에 대한 어려움이 있었다.

스크립트를 배우기 시작한 단계에서 필요한 슬라이드 구현을 하기 위해 적용을 해보았는데 작동이 되지 않았다.

구글링과 배운 것의 응용을 토대로 여러가지 시도를해보았더니 구현이 되었고 큰 성취감을 느낄 수 있었다.

Publishing Guide

Open Graph |

오픈그래피 적용 meta property="og" 적용

Favicon |

5개의 이미지와 1개의 JSON 파일로 파비콘 최적화 32x32 예전 브라우저 svg 최신 브라우저 / 다크모드 지원 apple-touch-icon 180x180 애플기기 192x192 홈스크린 manifest(JSON) PWA 및 안드로이드 기기 512x512 PWA 로딩시 스플래시 스크린용

HTML5 |

W3C기준 meta[name:viewport]설정 header / main / footer를 이용하여 영역 분리 img태그 사용시 alt값 필수 작성 a태그 사용시 빈 링크는 #으로 통일

CSS3 |

첫줄에 인코딩 선언 @charset "UTF-8"; 주석을 이용하여 영역별 css 구분 모든 속성 선언 끝에 세미콜론; 작성

파일명 |

웹페이지는 첫 페이지는 index.html로 통일 알파벳, 숫자, 언더바_ 조합하여 파일명 설정

폴더 구조 |

css 공통으로 연결된 font.css와 menu.css 담겨있는 폴더 fonts 웹폰트를 제외한 폰트 파일이 담겨있는 폴더 images 공통으로 연결된 이미지가 담겨있는 폴더 script 공통으로 연결된 .js 파일이 담겨있는 폴더
